# Supported Compilers

Following compilers are backed up by the GitHub Action [workflows](https://github.com/silverqx/TinyORM/tree/main/.github/workflows) (CI pipelines), these workflows also include more then 975 unit tests ðŸ˜®.

<div id="supported-compilers">

#### `Windows >=10` {#windows-10}

- MSVC 2019 >=16.9
- MSVC 2022 >=17
- MSYS2 UCRT64 GCC 10.2 - 11.3
- MSYS2 UCRT64 Clang 12 - 14
- clang-cl >=14 with MSVC 2019/2022 <small class='darker'>(without CI)</small>

#### `Linux`

- GCC 10.2 - 11.3
- Clang 11 - 14

</div>

:::tip
You can compile TinyORM with the MSVC 2022 even if Qt doesn't provide binaries for the MSVC 2022, you can link against Qt MSVC 2019 binaries without any limitations.
:::
